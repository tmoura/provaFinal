Componente 3

Estilize a tabela conforme ilustra a Figura 3. Contudo, considere os seguintes itens:

  • No arquivo html, crie a tabela usando o conteúdo comentado com <!-- Dados Tabela -->
  • Use o nome: “componente3” para o class da tag “table”
  • No css, estilize a tabela com a largura de 49%, e margem de .5%
  • O alinhamento das colunas deve ser preservado
  • Cada célula deve possuir recuo interno de .4rem .5rem, e borda de 1px solid #707070
  • Coloque bordas na tabela com espessura de 2px, cor preta e radius de 10%
  • A primeira linha deve ter background lightblue
